# سيطرة الشرق الأوسط — لعبة استراتيجية أونلاين

## تشغيل محلي (تجربة سريعة)
```bash
cd public
python3 -m http.server 8000
```
ثم افتح: http://localhost:8000

بدون إعداد Firebase، اللعبة تعمل بـ"وضع تجربة محلي" (لاعب واحد بمتصفح واحد، الحالة تُفقد عند إعادة التحميل).

## تفعيل اللعب الأونلاين الحقيقي (متعدد اللاعبين)
عدّل `public/js/firebase-config.js` بمفاتيح مشروعك على Firebase. التفاصيل الكاملة في دليل النشر المرسل سابقاً (GitLab + Firebase).

## النشر على GitLab Pages
1. ادفع هذا المجلد كمشروع GitLab (الهيكل جاهز: `.gitlab-ci.yml` + `public/`)
2. `git add . && git commit -m "أول نسخة من اللعبة" && git push`
3. من Build → Pipelines انتظر ✅ ثم من Deploy → Pages يظهر رابط موقعك

## هيكل المشروع
```
public/
├── index.html          الصفحة الرئيسية
├── style.css            التصميم
└── js/
    ├── countries-data.js   بيانات الدول (لا تُعدَّل يدوياً)
    ├── firebase-config.js  مفاتيح Firebase (عدّلها أنت)
    ├── store.js            طبقة المزامنة (أونلاين/محلي)
    ├── game-logic.js        قواعد اللعبة (موارد، بناء، تجنيد، هجوم)
    └── game.js               ربط الواجهة بالمنطق
```

## آلية اللعب (ملخص)
- كل لاعب يدخل باسمه ويختار لوناً، ثم يدخل كود غرفة (شاركه مع أصدقائك للعب معهم في نفس الغرفة)
- اضغط على أي دولة غير مملوكة لتستحوذ عليها
- ابنِ مباني (قاعدة دبابات، مطار، مرفأ...) لفتح وحدات أقوى
- جنّد وحدات عسكرية، ثم اهجم دولاً مجاورة للتوسع
- الموارد تتجدد تلقائياً كل 20 ثانية (نظام "تِك" حي بدل الأدوار الصارمة، لتسهيل اللعب الأونلاين)

## معروف كتحسينات مستقبلية (لم تُنفَّذ في هذه النسخة الأولى)
- سقف/حد أقصى لعدد الدول التي يملكها لاعب واحد
- تمييز الوحدات البحرية بشرط "دولة ساحلية"
- نظام أدوار متزامنة صارم (بدل التِك الحي) إن رغبت بإيقاع أبطأ
- قواعد أمان Firestore (منع أي شخص من تعديل غرفة لا ينتمي لها) — مهم قبل الإطلاق العلني الحقيقي
